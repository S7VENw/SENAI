POSITIONS

No que diz respeito a controle de posições dos elementos HTML, existem várias formas para alcançar tal resultado (positions, display, float), hoje iremos abordar sobre as variações da propriedade position.

Position é um módulo do CSS que define como ficam as posições dos elementos relativos e absolutos da página, fazendo referência aos valores atribuídos quanto ao top, bottom, right e left. Esta propriedade funciona perfeitamente nos navegadores desde a versão 2.1 do CSS, onde o seu foco é apenas o de posicionar pequenos detalhes do layout (para diagramar utilize Flexbox e Grid).

A propriedade em questão pode assumir 5 valores diferentes: Static, Relative, Absolute, Fixed e Sticky. Segue explicação de cada uma:

CSS Position Static:

Esse é o valor default de todo elemento HTML, ou seja, ele vai seguir o fluxo comum da sua página. Para declará-lo faça: 

                    Seletor {
                    position: static;
                   }

CSS Position Relative

Utilizando o position Relative o elemento passa a aceitar as propriedades Top, Bottom, Left e Right. Com elas você pode alterar o posicionamento do elemento. Ex:

HTML: 
                  <div>
                      <h1>Sou um h1</h1>
                      <h2>Sou um h2</h2>
                    </div>
CSS: 
                  div { background: #eee; margin: 200px 200px; width: 300px; height: 300px; }
                h1 { position: relative; top: 20px; left: 20px; }
                h2 { top: 20px; left: 20px; }

CSS Position Absolute

O position Absolute é um grande quebra galho no CSS. Com ele você pode posicionar qualquer elemento de acordo com o elemento pai que tenha um position diferente de static. 

                  <div>
                      <h1>
          
                          <span></span>
          
                      </h1>
          
                      <p></p>
                  </div>

O elemento DIV é pai dos elementos H1, P e SPAN. Os elementos H1, P e SPAN são filhos do elemento DIV. Os elementos H1 e P são irmãos por estarem no mesmo nível. E o elemento SPAN é filho do elemento H1 e também do elemento DIV.

HTML:

                  <div>
                      <h1>Sou um h1</h1>
                      <h2>Sou um h2</h2>
                    </div>

CSS:

                  div { background: #eee; margin: 50px;
                              width: 300px; height: 300px; position: relative;}
                h1 { position: absolute; top: 20px; left: 20px;
                              outline: 1px solid #444; }
                h2 { outline: 1px solid #f90; }

Observação: podemos utilizar tanto medidas positivas quanto negativas, e assim, posicionar os elementos em locais que antes seriam impossíveis de serem alcançados.

CSS Position Fixed:

O position fixed se comporta de maneira semelhante ao absolute, deixando de fazer parte do fluxo comum da página. Porém a grande diferença dele é que ele passa a se referenciar ao window do seu navegador, ou seja, a área que aparece para o usuário independente de barra de rolagem.

HTML:

                  <div>
                      <h1>Sou um h1</h1>
                      <h2>Sou um h2</h2>
                    </div>

CSS:

                  body { background: #000; }
              div { background: #f90; width: 300px; height: 1500px; position: relative; }
              h1 { position: fixed; top: 0; left: 20px; right: 20px;
                   outline: 1px solid #444; background: #fff;  }
              h2 { position: fixed; bottom: 0; left: 20px; right: 20px;
                  outline: 1px solid #444; background: #fff;}
